## features:

- user can login
- user can see the most state of the art components needed for building their dream pc
- user can go to pc builder option and select their required components to build their pc
- user can also see if there are good reviews for a specific component
- user can also see detailed specs of any product

## Run instruction:

- clone using: git clone <clone url>
- to install dependencies write: yarn install
- to run write: yarn dev
- after that you are good to go
